PythC is an online application dedicated to teach people programming for free.

We do this through a lesson based system to teach users different programming concepts.

Above that people can view helpful videos and test their knowledge by coding projects for the lessons. Finally, users can create lesson for other users to take and learn from.

Through PythC users can access a never ending database of lessons dedicated to teach them everything they need to know about programming.
